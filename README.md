# KBTU Lost & Found 🔍

Digital platform for students and staff of Kazakh-British Technical University to recover lost items.

## Tech Stack
* Frontend: Angular 17+
* Backend: Django REST Framework
* Database: SQLite (Development) / PostgreSQL
* Auth: JWT (JSON Web Token)

## Project Description
A specialized notice board where KBTU students can:
1. Post found items with descriptions and locations.
2. Search for lost items by categories (Keys, Electronics, etc.).
3. Claim ownership by sending a request with a description of unique features.
4. Manage their own listings and verify claims from other users.
